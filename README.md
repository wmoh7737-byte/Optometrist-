# مدقق النظر — تطبيق العناية بالعين

تطبيق Flutter لحماية العين أثناء استخدام الجوال، يشمل:
- تذكير دوري بقاعدة 20-20-20
- فلتر ضوء أزرق
- تحذير اهتزاز عند اقتراب الشاشة من الوجه (حساس القرب)

## طريقة التشغيل

1. تأكد إنه عندك Flutter SDK مثبت:
   https://docs.flutter.dev/get-started/install

2. من داخل مجلد المشروع:
   ```bash
   flutter pub get
   flutter run
   ```

3. لتجربة تحذير الاقتراب، لازم تشغّل التطبيق على **جهاز حقيقي** —
   المحاكيات (Emulator/Simulator) عادة ما تدعم حساس القرب الحقيقي.

## هيكلية المشروع

```
lib/
  main.dart              # نقطة الدخول + شريط التبويبات
  theme.dart              # الألوان والخطوط
  app_state.dart          # حالة الإعدادات والإحصائيات (مع الحفظ محلياً)
  services/
    proximity_service.dart  # الاستماع لحساس القرب + الاهتزاز
  screens/
    home_screen.dart        # الشاشة الرئيسية والمؤقت
    settings_screen.dart    # شاشة الإعدادات
    alert_screen.dart       # شاشة تحذير الاقتراب
```

## ملاحظات مهمة قبل الإطلاق

- **iOS**: حساس القرب فيه محدود بالوصول من طرف أبل. إذا حسيت إنه مو
  دقيق كفاية، فكّر تستخدم الكاميرا الأمامية مع ARKit/Face detection
  كبديل أدق (يحتاج مجهود إضافي).
- **فلتر الضوء الأزرق**: هاي النسخة فيها التفعيل/الإيقاف بس، لسا ما
  فيها تطبيق فعلي لطبقة اللون فوق الشاشة — تقدر تضيفها بسهولة عن طريق
  `Overlay` شفاف بلون كهرماني فوق كل الشاشات.
- **وقت الشاشة الفعلي**: الرقم بالإحصائيات حالياً مجرد قيمة تجريبية،
  لتتبع حقيقي بتحتاج قراءة استخدام النظام (Usage Stats على أندرويد
  أو Screen Time API على iOS) وهذي بتحتاج صلاحيات إضافية.
- قبل النشر لازم تضيف نص "سياسة الخصوصية" توضح إنك ما بتخزن ولا
  ترسل أي بيانات حساسة (خصوصاً لو استخدمت الكاميرا لاحقاً).

## الخطوة الجاية المقترحة

1. جرب التطبيق على جهاز حقيقي وشوف كيف حساس القرب بيتصرف
2. اضبط قيمة `Duration(seconds: 5)` بملف `proximity_service.dart`
   حسب تجربتك (فترة التهدئة بين تحذير وآخر)
3. أضف تتبع وقت الشاشة الحقيقي
4. صمم أيقونة التطبيق (App Icon) وشاشة البداية (Splash Screen)
